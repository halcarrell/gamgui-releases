# GAM7 Admin - Release Downloads

This repository contains release files for GAM7 Admin.

## Download Latest Version

Visit the [Releases](https://github.com/halcarrell/gamgui-releases/releases) page to download the latest version.

## Installation

1. Download the DMG file from the latest release
2. Open the DMG
3. Drag the app to Applications
4. Launch the app

---

**Note:** This repository only contains release files. Source code and documentation are in the private repository.
