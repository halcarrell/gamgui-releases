# GAM Companion — Releases

Public download & auto-update channel for **GAM Companion**, the macOS GUI for GAM
(Google Apps Manager) / Google Workspace administration. Source code lives in a
private repository; this repo hosts **signed, notarized release artifacts only**.

## Download the latest

**[Download GAM Companion (.dmg)](https://github.com/halcarrell/gamgui-releases/releases/latest/download/GAM7-Admin-Latest.dmg)** — this link always points to the newest release.

- **Latest version:** `v2.0.7` (2026-05-30)
- macOS universal (Apple Silicon + Intel), signed & notarized (Developer ID: Hal Carrell).
- Already installed? The app updates itself via the built-in updater.

### Install
1. Download the `.dmg` above.
2. Open it and drag **GAM Companion** to **Applications**.
3. Launch it, then connect GAM and sign in with Google.
